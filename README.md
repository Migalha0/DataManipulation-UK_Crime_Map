# England crime map

This tool facilitates the visualisation of data provided by the UK Police overlayed on top of a map of England + Wales

-

## Tools used

- Vite

## To run locally

1. Install dependencies:

```bash
npm i
```

2. Start the development server:

```bash
npm run dev
```
